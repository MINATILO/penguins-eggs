Penguin's eggs

Questa remix può essere utilizzata così com'è - live - oppure installata sul vostro computer.

Utenti

Sia l'utente live che l'utente root, se non diversamente specificato sono impostati (user/password) in questo modo:

  • live/evolution
  • root/evolution

    • Utilizzo della remix come live

    L'utilizzo di questo sistema in forma live non comporta, - di massima - pericoli per i dati sul disco rigido.

    Fate naturalmente attenzione a non cancellare i vostri dati.

    Installazione

    L'installazione può essere fatta sia con l'installatore grafico (opzione consigliata) oppure può avvenire da terminale.

    Calamares - installazione grafica

    Calamares è un programma di installazione GUI molto sofisticato, sicuro ed è l'opzione consigliata per la maggior parte degli utenti.

    Può permettere la coesistenza di questo sistema con altri precedentemente installati

    eggs install - Installazione da terminale

    eggs hatch è un installatore da terminale molto semplice e scarno, che NON permette l'installazione del sistema a fianco di un altro preesistente, anzi CANCELLA interamente il sistema precedente.

    Il vantaggio di eggs install rispetto a calamares è la maggiore leggerezza che gli consente di eseguire l'installazione più velocemente su macchine con poca memoria RAM.

    Se ne consiglia l'uso SOLO nel caso l'installazione grafica risultasse eccessivamente lento e, quindi, non usabile.
